# sg-replay-parser

**Русский** · [English](README.en.md)

> ⚠️ **Устарело и заморожено.** Репозиторий не развивается и не поддерживается.

Прежний генератор статистики для сообщества [Solid Games](https://sg.zone)
(ArmA 3): набор задач, забиравших и разбиравших данные реплеев с
`sg.zone/replays`.

Заменён на **[replay-parser-2](https://github.com/Afgan0r/replay-parser-2)** —
используйте его. Сюда новые изменения не вносятся; история и код оставлены
только для справки.
